1.
cv::Mat : 행/열을 표현할 수 있는 클래스
- frame, shot, gray, color128
- cols(행), rows(열), type(CV_8UC1 = 흑백, CV_8UC3 = 컬러)
--------------------------------------------------------------
2.
cv::VideoCapture : 카메라 또는 동영상 파일로 정지된 영상 프레임을 받아올 수 있는 클래스
- open(int index) -> 카메라 열기
- isOpend() -> 카메라 열기 확인
- release() -> 장치 및 파일 자원 해제(카메라 닫기)
- operator >> -> 다음 프레임을 읽기(여기선 cap >> frame 부분)
--------------------------------------------------------------
3.
cv::CascadeClassifier : Haar Cascade 기반 얼굴 탐지기(정면 얼굴모델: haarcascade_frontalface_default.xml) 사용
- load(path) : XML 학습 모델 로드
- empty() : 로드 여부 확인
- detectMultiScale() : 얼굴 후보 탐색 -> Rect 목록(vector<Rect>) 반환
--------------------------------------------------------------
4.
cv::Rect : 사각형(ROI, 얼굴 영역) 표현 클래스
- x, y : 좌/상단 좌표
- width, height : 폭, 높이
- cv::Point(x, y) : 좌표
- cv::Size(width, height) : 넓이
- area() : 면적
--------------------------------------------------------------
5.
cv::cvtColor(src, dst, cv::COLOR_BGR2GRAY) : 색상 변환
- src : 입력
- dst : 출력
- 변환 코드 : BGR2GRAY(흑백), BGR2RGB(RGB 변환 -> Mat은 BGR, 프로그램은 RGB라 변환 필요)
--------------------------------------------------------------
6.
cv::equalizeHist(src, dst) : 히스토그램 평활화(평면적인 이미지의 대비시켜 선명한 이미지를 획득)
- src : 입력
- dst : 입력
- 여기서는 (gray, gray)로 사용
--------------------------------------------------------------
7. cv::rectangle(image, rect, color, thickness)  : 사각형 그리기(시각화 용도)
- image : 대상 이미지(여기선 Mat)
- rect : 사각형(Rect)
- color : 색상(Scalar)
 cv::Scalar(0,255,0) : 색상 설정(초록색)
- thickness : 선 두께(int)
--------------------------------------------------------------
8. cv::imencode() : 인코딩
: Mat 이미지를 PNG/JPEG 등으로 인코딩
